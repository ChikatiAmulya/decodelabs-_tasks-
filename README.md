# DecodeLabs Artificial Intelligence Projects

Welcome to my AI Projects Repository! This repository contains the projects completed during the **DecodeLabs Artificial Intelligence Industrial Training Program**. Each project focuses on applying Artificial Intelligence concepts through practical implementation using Python.

##  About

The purpose of this repository is to showcase my learning journey in Artificial Intelligence, Machine Learning, Recommendation Systems, and Python Programming through hands-on projects.

##  Projects Included

### 1. AI Chatbot

A rule-based chatbot that interacts with users and provides predefined responses based on user input.

**Key Features**

* Interactive conversation
* Rule-based response generation
* User-friendly interface
* Python fundamentals implementation

### 2. AI Classification System

A machine learning project that predicts and classifies data into different categories using classification algorithms.

**Key Features**

* Data preprocessing
* Model training and testing
* Prediction generation
* Performance evaluation

### 3. AI Recommendation System

A recommendation engine that suggests courses based on user interests using similarity matching and recommendation logic.

**Key Features**

* User preference analysis
* Similarity score calculation
* Pattern matching
* Personalized recommendations
